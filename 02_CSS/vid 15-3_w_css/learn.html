3 ways to add css in html

1.Inline CSS
-by this way attribute style is added to the element and commands of css are given in the same tag
-Avoid this way mostly

2.Internal CSS
-by this way an style tag is used in head and then commands are given in that tag by choosing selectors
-3rd way is more preferable than this

3.External CSS
-instead of adding css data in html the separate .css file is created and linked by using link tag to html file
-It is the most recommanded way to add CSS
